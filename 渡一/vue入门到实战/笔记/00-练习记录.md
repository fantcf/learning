### 1） css相关问题：
- 问题描述：针对外层div进行样式设置的时候，没有作用到img身上；
```html
<div class="avatar-container">
    <img src="https://www.baidu.com/pic" alt="" />
</div>
<style>
    /*错误写法：*/
    .avatar-container {
        border-radius: 50%;
        object-fit: cover;
    }
    /*正确写法*/
    .avatar-container{
        img {
            border-radius: 50%;
            object-fit: cover;
        }
    }
</style>
```
